Our application was built using the MERN (MongoDB, Express.js, React.js, and Node.js) stack.

Front-End Technology

    - Apollo Client
    - Chakra UI
    - PetFinder API

Back-End Technology
    - Node.js & Express
    - Apollo & GraphQL
    - MongoDB & Mongoose
    - JWT
    - Bcrypt
